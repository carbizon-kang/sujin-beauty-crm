# 프로젝트 가이드 (CLAUDE.md)
이 파일은 Claude가 이 프로젝트의 목적과 규칙을 이해하도록 돕는 가이드라인입니다.

## 💄 프로젝트 개요
- **목적**: 뷰티 매장(미용실, 피부관리 등) 고객 관리 및 시술 이력 자동화 시스템 구축
- **주요 기능**: 
  - 고객 기본 정보(성함, 연락처, 특징) 등록 및 검색
  - 시술 히스토리 기록 (날짜, 시술 종류, 사용 제품, 단가)
  - 예약 현황 관리 및 방문 주기 알림 분석
  - 일간/월간 매출 통계 및 세무 보고용 데이터 생성
- **사용 도구**: Python, Streamlit, Pandas, Claude Code

## 📋 업무 규칙 (Rules)
- 모든 코드는 파이썬(Python)을 기반으로 작성하며, UI는 Streamlit을 활용합니다.
- 코드 내 주석과 모든 설명은 **한국어**로 친절하게 작성합니다.
- 데이터 처리는 가급적 **Pandas** 라이브러리를 활용하여 엑셀과 호환되게 합니다.
- 고객의 개인정보(연락처 등)가 포함되므로 데이터 보안과 백업을 최우선으로 합니다.
- 복잡한 로직이나 UI 수정 시 이전 버전을 아티팩트(Artifacts)로 관리합니다.

## 📁 하위 폴더 구조
- **/data**: 고객 명단 및 시술 이력 엑셀(`customers.xlsx`) 파일 보관
- **/utils**: 매출 계산, 세무 신고용 로직 파이썬 파일 보관
- **/assets**: 매장 로고 및 UI용 이미지 파일 보관
